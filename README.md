<h1>Jogo do número secreto</h1>

<h2> 📖 Sobre</h2>
<p>O Jogo do Número Secreto é uma aplicação web simples desenvolvida com HTML, CSS e JavaScript, utilizada para praticar conceitos de lógica de programação.

O objetivo do jogo é descobrir qual número foi sorteado pelo sistema dentro de um intervalo definido. A cada tentativa, o jogo informa se o número secreto é *maior ou menor* que o número digitado, ajudando o jogador a chegar na resposta correta.

O projeto também utiliza *síntese de voz*, permitindo que as mensagens exibidas na tela sejam narradas automaticamente.</p>

## ⚙️ Funcionalidades

- 🎲 Geração aleatória de um número secreto entre **1 e 100**
- 🔁 Sistema de tentativas para adivinhar o número
- ⬆️ ⬇️ Dicas informando se o número secreto é **maior ou menor**
- 🔊 Narração automática das mensagens usando **ResponsiveVoice**
- 🔄 Botão para **reiniciar o jogo**
- 🧠 Controle para evitar repetição de números já sorteados
